= Wordle Game Service

== Overview

The Game Service is a microservice component of the Wordle application that handles game logic and word validation 

== Word Dictionary Sources

1. **Primary**: GitHub repository 
2. **Fallback**: Local cache
3. **Emergency**: Hardcoded word list (500+ words)

== Endpoints

[cols="1,2,3"]
|===
|Method |Endpoint |Description 

|POST |`/game/new` |Create a new game 
|POST |`/game/:gameId/guess` |Submit a word guess 
|GET |`/game/:gameId` |Get current game state 
|GET |`/health` |Service health check 
|===

== Local Deployment
[source,bash]
----

./run-local.sh

# Manual execution  
npm install
npm run build
npm start

# Run tests
npm test
npm run test:coverage

curl http://localhost:3002/health
open http://localhost:3002/api-docs
